Ты — эксперт-сметчик. Проанализируй данные строительного элемента.

## ДАННЫЕ ЭЛЕМЕНТА:
{data_str}

## ЗАДАЧА:
Выдели и раздели ВСЕ характеристики элемента на ТРИ основные группы:

1. **РАЗМЕРЫ** - только размерные характеристики с числами:
   - длина, ширина, высота, толщина, глубина
   - площадь, объем
   - параметры с единицами измерения (мм, м, м², м³, литры)
   - ЕСЛИ РАЗМЕРОВ НЕТ - оставь пустой объект {{}}

2. **МАТЕРИАЛ** - из чего сделан элемент:
   - ОПРЕДЕЛИ МАТЕРИАЛ из поля "Имя" и других полей
   - Название материала должно быть понятным
   - количественные характеристики материала (с числами):
     * марки бетона: В30, В25, В20
     * водонепроницаемость: W6, W8
     * морозостойкость: F150, F200
     * толщина: 100мм, 200мм
     * плотность: 2400 кг/м³
   - качественные характеристики материала (без чисел):
     * состав: из каких слоев
     * тип: рулонный, плитный
     * вид: монолитный, сборный

3. **ОПИСАНИЕ ЭЛЕМЕНТА** - характеристики самого элемента:
   - тип элемента: стена, колонна, термовкладыш
   - технология: монолитная, сборная
   - другие описательные характеристики

## ФОРМАТ ОТВЕТА (ТОЛЬКО JSON, БЕЗ ДРУГОГО ТЕКСТА):

{{
  "размеры": {{
    "параметр": "значение с единицей"
  }},
  "материал": {{
    "название": "название материала",
    "количественные_характеристики": {{
      "параметр": "значение"
    }},
    "качественные_характеристики": {{
      "параметр": "значение"
    }}
  }},
  "описание_элемента": {{
    "параметр": "значение"
  }}
}}

## ВАЖНЫЕ ПРАВИЛА:
1. Если размеров нет - "размеры" должен быть пустым объектом {{}}
2. Не используй фразы "не указаны", "нет данных" - просто оставляй пустые объекты
3. Если нет количественных характеристик - оставляй пустой объект {{}}
4. Если нет качественных характеристик - оставляй пустой объект {{}}
5. ВСЕ значения должны быть строками
6. Не добавляй то, чего нет в данных
